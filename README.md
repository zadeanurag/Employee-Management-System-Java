# Employee Management System (Core Java)

## Description
A console-based Java application developed using Core Java concepts such as OOPs and Collection Framework.

## Features
- Add, Update, Delete, Search Employees
- Uses HashMap and ArrayList
- Exception handling for input validation
- Menu-driven console interface

## Technologies Used
- Core Java
- OOPs
- Collection Framework

## How to Run
1. Open the project in Eclipse or any Java IDE
2. Run `EmployeeManagementApp.java`
